**Exam Details**:

#### Time Management
- **Allocate time**: ~1.4 minutes per question
- **Answer easy questions first**: Build confidence
- **Flag difficult questions**: Return to them later
- **Don't spend too long on one question**: Move on if stuck
- **Use process of elimination**: Eliminate wrong answers

#### Question Strategies
- **Read carefully**: Understand what's being asked
- **Look for keywords**: "Best", "Most", "Primary", "Main"
- **Consider all options**: Don't jump to conclusions
- **Use context clues**: Information in the question
- **Trust your preparation**: Don't second-guess yourself

#### Multiple Response Questions
- **Read instructions carefully**: "Select TWO" vs "Select ALL"
- **Consider each option independently**: Don't assume relationships
- **Use process of elimination**: Eliminate wrong answers
- **Check your selections**: Ensure you've selected the right number

### After the Exam

#### Immediate Actions
- **Don't discuss specific questions**: Maintain exam integrity
- **Review your performance**: Note areas for improvement

## üìù Practice Questions by Domain

### Domain 1: Cloud Concepts (24%)

#### Question 1
**What is the main benefit of cloud computing over on-premises infrastructure?**
- A) Lower upfront costs
- B) Better security
- C) More control
- D) Faster performance
- **Answer**: A - Cloud computing eliminates upfront infrastructure costs

#### Question 2
**Which AWS service provides the most cost-effective option for long-term data archival?**
- A) Amazon S3 Standard
- B) Amazon S3 Glacier Deep Archive
- C) Amazon EBS
- D) Amazon EFS
- **Answer**: B - S3 Glacier Deep Archive is designed for long-term archival

#### Question 3
**What is the primary purpose of the AWS Well-Architected Framework?**
- A) To reduce costs
- B) To improve security
- C) To provide architectural best practices
- D) To increase performance
- **Answer**: C - The framework provides architectural best practices

### Domain 2: Security and Compliance (30%)

#### Question 4
**Which AWS service provides centralized access management?**
- A) Amazon S3
- B) AWS IAM
- C) Amazon EC2
- D) Amazon RDS
- **Answer**: B - IAM provides centralized access management

#### Question 5
**What is the Shared Responsibility Model?**
- A) AWS is responsible for everything
- B) Customer is responsible for everything
- C) AWS and customer share responsibilities
- D) Third-party vendors are responsible
- **Answer**: C - AWS and customer share responsibilities

#### Question 6
**Which service provides threat detection and monitoring?**
- A) AWS Config
- B) Amazon GuardDuty
- C) AWS CloudTrail
- D) Amazon Inspector
- **Answer**: B - GuardDuty provides threat detection

### Domain 3: Cloud Technology and Services (34%)

#### Question 7
**Which AWS service provides serverless compute?**
- A) Amazon EC2
- B) AWS Lambda
- C) Amazon ECS
- D) Amazon EKS
- **Answer**: B - Lambda provides serverless compute

#### Question 8
**What is the main difference between Amazon S3 and Amazon EBS?**
- A) S3 is block storage, EBS is object storage
- B) S3 is object storage, EBS is block storage
- C) S3 is file storage, EBS is object storage
- D) S3 is block storage, EBS is file storage
- **Answer**: B - S3 is object storage, EBS is block storage

#### Question 9
**Which service provides global content delivery?**
- A) Amazon Route 53
- B) Amazon CloudFront
- C) AWS Direct Connect
- D) Amazon VPC
- **Answer**: B - CloudFront provides global content delivery

### Domain 4: Billing, Pricing, and Support (12%)

#### Question 10
**Which pricing model offers the highest discount for predictable workloads?**
- A) On-Demand
- B) Reserved Instances
- C) Spot Instances
- D) Dedicated Hosts
- **Answer**: B - Reserved Instances offer up to 75% savings

#### Question 11
**What is the main benefit of AWS Organizations?**
- A) Lower costs
- B) Consolidated billing and volume discounts
- C) Better security
- D) Faster performance
- **Answer**: B - Organizations provides consolidated billing and volume discounts

#### Question 12
**Which support plan includes a Technical Account Manager?**
- A) Basic Support
- B) Developer Support
- C) Business Support
- D) Enterprise Support
- **Answer**: D - Enterprise Support includes a TAM

## üéØ Practice Exam Tips

### Question Analysis
1. **Read the entire question**: Don't skip any words
2. **Identify the key concept**: What AWS service or concept is being tested?
3. **Look for qualifiers**: "Best", "Most", "Primary", "Main"
4. **Consider the context**: What scenario is described?
5. **Eliminate wrong answers**: Use process of elimination

### Common Question Patterns
- **Service selection**: Which service is best for X?
- **Feature comparison**: What's the difference between X and Y?
- **Use case matching**: What service would you use for X scenario?
- **Best practices**: What's the best way to do X?
- **Troubleshooting**: How would you solve X problem?

### Time Management
- **First pass**: Answer all easy questions
- **Second pass**: Return to flagged questions
- **Final review**: Check your answers
- **Don't change answers**: Unless you're certain
